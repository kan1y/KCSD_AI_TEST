# Инструкции для ИИ-агента (Системный промпт)

Этот репозиторий представляет собой базу знаний бизнес-процессов АО "Центральный депозитарий ценных бумаг". 

## Правила работы с репозиторием (Реляционная структура)
1. **Связывание по ключу:** Данные нормализованы. Для сборки полного контекста о процессе используй уникальный идентификатор (например, `M 8.1`, `B 2.2`). Ищи файлы с соответствующим именем (`M_8_1.md`, `B_2_2.bpmn`) в директориях:
   - `21-Process-Cards/` — для понимания входов, выходов и участников.
   - `22-Processes/` — для понимания пошагового алгоритма.
   - `23-Scoring/` — для оценки критичности и ценности.
   - `24-Metrics/` — для анализа узких мест (Cycle Time, Wait Time).
   - `25-BPMN-Models/` — для анализа XML-схемы процесса (нотация BPMN 2.0).
2. **Контекст нормативной базы:** При анализе процессов строго руководствуйся документами из `10-Governance/`.
3. **Роль ОБА:** Отдел бизнес-анализа (ОБА) является владельцем архитектуры процессов и базы знаний.
4. **Язык:** Все выводы и анализ генерируются на русском языке с сохранением оригинальных английских терминов.
